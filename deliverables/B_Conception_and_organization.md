# Architecrure breakdown:
- Backend was handled with Node.Js and Expresx, MongoDB for DB and Gemini API for AI API.
- Frontend was handled with React and Vite and Tailwind.
# Task breakdown and workflow:
- Started with the backend first, then I obtain an API key and set up my API service. With that done, I moved to models then routes as well as implementing the authentication middleware. I then tested the backend via postman to ensure working condition.
- When backend had been finished, I moved on to frontend. I set up the node modules I needed, configure Tailwind. The chat UI and login, signup page was implemented as I go.
- Throughout the project, I used Notion as the tool for organising and keeping tabs on the project.
# Feedback on self development:
- During the project, I encountered issues with my Tailwind and Node version. I resolved this issue by downloading nvm-set.exe to freely use the node versions. Alongside that, with Tailwind, with v4 it didn't support the tailwind init line I was trying to use, so in the end, I had to use tailwindv3.
