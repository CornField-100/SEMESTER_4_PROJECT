# This project is a "Niche" project:
- "My Lang Buddy" is a chatbot to help students practice language. The key innovative feature is the 2 modes an user can interact with the bot: Teacher and Peer.
- Why it is "Niche": while other chatbots(GPT, Gemini, ...) exist, my project usage of dual-role correction and peer focused support is less common in the chatbot ecosystem.
