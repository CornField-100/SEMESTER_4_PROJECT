# My Lang Buddy: A Simple AI Chatbot
## Overview
1. Concept
2. Techonologies used
3. Main AI API
### 1. Concept:
The project is a simple AI chatbot, complete with front end and backend. "My Lang Buddy" is a chatbot aimed to help students with language studies, featuring 2 modes: "teacher" for studies and "peers" for general conversation

### 2. Technologies used:
- Backend built using node JS
- Frontend built using React + Vite
### 3. AI API used:
- The project was done using Google Gemini, specifically gemini-2.5-flash.
